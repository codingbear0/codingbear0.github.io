- 현재[^1] 태그는 아래와 같이 설정:
	| | |
	| --- |  --- |
	| @critical | 긴급하고 중요한 일 |
	| @urgent | 긴급하지만 중요하지 않은 일 |
	| @important | 긴급하지 않지만  중요한 일 |
	| @chore | 긴급하지도 않고 중요하지도 않은 일 |
	| @10min | 10분 안에 할 수 있는 일 |
	| @deferred | 당장 처리할 수 없어 미룬 일 |
	| @pending | 외부 상황에 의해 대기 중인 일 |
